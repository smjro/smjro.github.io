numpy配列の型変換

programming - 01/29/2017
python
numpy

pythonを使用しているとnumpyを用いることが多いと思いますが、 numpyでの型の変換の仕方を知らなかったので、ここに残しでおきます。

型の変換

numpyの多次元配列（ndarray）の型（dtype）を変更するにはastypeを使います。

>>> import numpy as np
>>> x = np.array([-1.0, 1.0, 2.0])
>>> x.dtype
dtype('float64')
>>> int_x = x.astype(np.int)
>>> int_x.dtype
dtype('int64')
>>> int_x
array([-1,  1,  2])
